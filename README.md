# A collection of Python projects
A collection of my small Python projects and practical scripts. 

# Description
Mini Python Projects is a small educational repository with Python examples, experiments, and practice problems. It contains mini-projects of varying complexity, from simple scripts to small algorithms and test ideas (though the simpler ones are often more common). Each project is located in a separate folder and is a standalone example created for practicing and learning the language. The repository is not intended to be complete or exhaustive; it is a working environment for learning, experimenting, and saving useful code snippets. Almost all projects are written in pure Python without external libraries. I tried to create them myself, rarely using AI (except when exploring new libraries or new language features). The repository is distributed under the MIT license and is free to use, modify, and expand upon.

# Directory structure
Each directory will contain:
- Source code as a .py file;
- readme.md with an image of what the program does (optional)
